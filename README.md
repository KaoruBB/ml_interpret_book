「機械学習を解釈する技術〜予測力と説明力を両立する実践テクニック」のサンプルコードです。

## 動作環境

Pythonのバージョンは3.8を、パッケージの管理にはpoetryを利用しています。
サンプルコードを`git clone`でダウンロードし、フォルダ直下で`poetry install`を実行すると、本書と同じ環境を構築することが出来ます。
